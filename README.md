#### Don't forget to add this to your PYTHONPATH

Before running the file `design_patch.py` or `design_patch.ipynb`, Update your **PYTHONPATH** by running this command,

```text
export PYTHONPATH=$PYTHONPATH:/path/to/patch_util
```
